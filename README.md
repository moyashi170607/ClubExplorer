# ケモミミ娘がオススメの部活を教えてくれるアプリ

### ～ケモミミ娘と一緒に岡山大学で君にピッタリの部活を探しに行こう！～

## 概要
このソフトはケモミミ娘からの質問に答えるとあなたにオススメの岡山大学の部活を紹介してくれるアプリです。
新入生の方はぜひご利用ください。
